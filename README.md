# EduAI - AI Student Assistant (Integrated with OpenAI)
A beautiful, state-of-the-art AI Student Assistant web application integrated with OpenAI features. Features an AI Tutor, Smart Notes Summarizer, AI Quiz Generator, Personalized Study Planner, and an AI Coding Mentor with built-in sandbox terminals for Python, JS, Java, and C++.

An advanced, interactive, and beautifully designed AI-powered study platform designed to serve as a personal tutor, study planner, coding mentor, and productivity coach for students.

---

## 🛠️ Required Tools & Prerequisites

Ensure you have the following tools installed on your local machine:

1. **Node.js** (v18.0.0 or higher recommended)
   - Download: [nodejs.org](https://nodejs.org/)
2. **npm** (comes packaged with Node.js) or **Yarn** / **pnpm**
3. **Modern Web Browser** (Chrome, Firefox, Edge, Safari)
4. **Code Editor** (e.g., VS Code)

---

## 🚀 How to Run Locally

Follow these steps to set up and run the application on your computer:

### 1. Install Dependencies
Navigate to the project root directory and run:
```bash
npm install
```

### 2. Start the Development Server
Run the following command to launch the local development environment:
```bash
npm run dev
```

### 3. Access the Application
Open your browser and navigate to:
```
http://localhost:5173
```

---

## 📦 Production Build

To build the application for deployment or hosting:

```bash
npm run build
```
This will generate a highly optimized single-file build inside the `dist/` directory.

## 🧠 Core Features Implemented

*   **Dashboard UI**: Beautiful glassmorphism UI showing study streaks, active hours, AI usage, upcoming exams, and smart suggestions.
*   **Smart AI Chat Tutor**: Chat with subjects like Physics, Biology, and Math, with quick actions such as *Summarize*, *Generate Quiz*, *Explain Simply*, and *Translate*.
*   **AI Notes Summarizer**: Organizes your study materials into folders with quick AI text highlights and summary utilities.
*   **AI Quiz Generator**: Supports MCQ, True/False, and Flashcard creation with real-time feedback and detailed brain-powered explanations.
*   **Personalized Study Planner**: Weekly dynamic scheduling with progress meters, calendar pickers, and AI-driven study recommendations.
*   **Coding Assistant**: Includes a full compiler UI for JavaScript, Python, C++, and Java, plus custom AI diagnostics (Explain, Debug, Optimize, Comment).
*   **Settings Preferences**: Theme switcher (Dark/Light), language selection, notification configuration, and tier plan upgrades.
